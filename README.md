# Todo Management Application

A Full Stack To-Do Management Application built with Spring Boot, React JS, and MySQL.

## Tech Stack
- **Backend:** Java 17, Spring Boot 3.3.4, Spring Data JPA, MySQL 8
- **Frontend:** React JS 19, React Router DOM, Axios, Vite

## Features
- Add, Update, Delete tasks
- View all tasks / View task by ID
- Mark task as Completed
- Search by Title and Priority
- Filter by Status (All / Pending / Completed)
- Delete confirmation dialog

## How to Run

**Backend**
- Create a MySQL database named `tododb`
- Update credentials in `application.properties`
- Run the Spring Boot application → starts at `http://localhost:8080`

**Frontend**
- Run `npm install` inside `todo-react-app` folder
- Run `npm run dev` → starts at `http://localhost:5173`
